# Programming Practice - Academic Labs 🚀

This repository contains a collection of practical exercises and projects completed as part of the "Programming Practice" course at Wroclaw University of Science and Technology. 

The main goal of this course was to gain hands-on experience with modern software engineering practices, clean code principles, and deployment automation.

## 🛠️ Tech Stack
* **Languages:** Python
* **Testing:** unittest

---

## 📂 Repository Structure & Topics Covered

The repository is divided into directories, each corresponding to a specific laboratory session:

### `01-tdd/` - Test-Driven Development (TDD)
This module focuses on practicing Test-Driven Development (TDD) in Python, featuring pair programming exercises (String Calculator Kata) and automated functional UI testing using Selenium.

### `03-refactoring/` - Code Review & Refactoring
